# Customer Service Prompts

## 1. Handle a Complaint

```
You're a customer service rep for [company name]. A customer just sent this complaint:

"[paste complaint here]"

Write a reply that:
- Acknowledges their frustration
- Apologises without being over the top
- Offers a clear solution or next step
- Keeps it under 150 words

Tone: friendly but professional. No corporate waffle.
```

## 2. Create FAQ Answers

```
I run a [type of business] called [company name]. Write FAQ answers for these questions:

1. [question 1]
2. [question 2]
3. [question 3]

Keep each answer to 2-3 sentences. Write like a real person — not a legal document.
```

## 3. Respond to a Refund Request

```
A customer wants a refund because [reason]. Our policy is [your policy].

Write a reply that:
- Thanks them for getting in touch
- Explains what we can do
- Gives clear next steps
- Stays warm but firm if we can't offer a full refund
```

## 4. Escalation Template

```
This support ticket needs to go to a manager. Summarise it in 3 bullet points:
- What the customer wants
- What we've already tried
- Why it needs escalating

Original conversation:
[paste conversation]
```

## 5. Live Chat Opening Messages

```
Write 5 different opening messages for a live chat widget on a [type of business] website.

Requirements:
- Each one under 20 words
- Sound human, not robotic
- One should mention a current offer: [offer details]
- Mix of questions and statements
```

## 6. Tone Adjustment

```
Rewrite this customer service email to sound [more friendly / more professional / more apologetic / more direct]:

[paste email]

Keep the same information but change the tone completely.
```
