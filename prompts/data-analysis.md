# Data Analysis Prompts

## 1. Spreadsheet Formula Helper

```
I have a spreadsheet with these columns: [list your columns].

I need a formula that [what you want to calculate/find].

I'm using [Google Sheets / Excel].

Give me the formula and explain what each part does in plain English.
```

## 2. Clean Up Messy Data

```
I have a dataset with these problems:
- [problem 1, e.g., dates in different formats]
- [problem 2, e.g., duplicate entries]
- [problem 3, e.g., missing values]

The data is in [Google Sheets / Excel / CSV].
Give me step-by-step instructions to clean it up, including any formulas I need.
```

## 3. Create a Dashboard Brief

```
I need a dashboard that shows [what you want to track].

Data sources: [where the data comes from]
Audience: [who will look at this]
Key metrics: [list 3-5 things to measure]
Update frequency: [daily/weekly/monthly]

Suggest the best chart types for each metric and explain why.
Tool: [Google Sheets / Excel / Power BI / other]
```

## 4. Explain This Data

```
Here's a summary of our [sales/marketing/support] data for [time period]:

[paste your data or key numbers]

Tell me:
1. What's going well
2. What's not going well
3. What I should do about it

Keep it simple — explain it like I'm not a data person.
```

## 5. Survey Analysis

```
I ran a survey with [X] responses. Here are the questions and a summary of answers:

[paste summary or raw data]

Give me:
- The 3 biggest takeaways
- Any surprising patterns
- What I should do next based on this data
- A short summary I can share with the team (under 200 words)
```

## 6. KPI Tracking Setup

```
I run a [type of business]. Help me set up KPI tracking.

Tell me:
- The 5-7 most important metrics I should track
- How to measure each one
- What "good" looks like for each (benchmarks)
- How often to check each one
- The simplest tool to track them in

Don't overcomplicate it — I'm a small business, not a corporation.
```

## 7. Competitor Data Comparison

```
I have data on my business and [X] competitors:

My numbers: [your key metrics]
Competitor 1: [their metrics]
Competitor 2: [their metrics]

Compare them and tell me:
- Where I'm ahead
- Where I'm behind
- What the gap looks like
- What to focus on first to close the gap
```
