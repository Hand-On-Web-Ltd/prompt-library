# Social Media Prompts

## 1. Weekly Content Calendar

```
Create a 5-day social media calendar for [company name], a [type of business].

Platforms: [LinkedIn/Instagram/Twitter/Facebook]
Goal this month: [awareness/leads/engagement]
Tone: [professional/casual/fun]

For each day, give me:
- Platform
- Post type (text, image idea, video idea, poll)
- The actual caption/copy
- Suggested hashtags (5-8)
```

## 2. LinkedIn Post

```
Write a LinkedIn post about [topic] for [your name], [your role] at [company].

Angle: [personal story / industry opinion / tip / behind the scenes]
Goal: [engagement/authority/leads]

Rules:
- Hook in the first line (make people click "see more")
- Short paragraphs (1-2 sentences each)
- End with a question or CTA
- No hashtag spam — max 3-5 relevant ones
- 150-200 words
```

## 3. Instagram Caption

```
Write an Instagram caption for a photo of [describe the photo].

Account: [company/personal brand]
Vibe: [fun/professional/inspirational/educational]

Include:
- An attention-grabbing first line
- The main message (3-4 sentences)
- A call to action
- 15-20 relevant hashtags (mix of popular and niche)
```

## 4. Twitter/X Thread

```
Turn this topic into a Twitter/X thread (5-7 tweets):

Topic: [your topic]
Audience: [who should care]

Rules:
- Tweet 1 = hook that makes people want to read on
- Each tweet should stand alone but flow as a story
- Last tweet = CTA or summary
- Keep each tweet under 250 characters
- No "thread 🧵" — that's overdone
```

## 5. Social Proof Post

```
Turn this client result into a social media post:

Client type: [industry]
What we did: [service]
Result: [outcome/numbers]

Write it as a short story, not a brag. Focus on the client's win, not ours.
Don't use the client's name — keep it anonymous unless told otherwise.
Platform: [LinkedIn/Instagram/Twitter]
```

## 6. Engagement Bait (The Good Kind)

```
Write 5 engagement posts for [platform] that aren't cringe.

Industry: [your field]
Goal: get comments and shares

Types to include:
- An unpopular opinion
- A "this or that" question
- A fill-in-the-blank
- A mini poll
- A hot take

Keep them genuine, not manipulative. No "like if you agree" nonsense.
```

## 7. Video Script (Short Form)

```
Write a script for a 30-60 second [Reel/TikTok/Short] about [topic].

Format:
- Hook (first 3 seconds — what stops the scroll)
- Main content (the meat)
- CTA (what to do next)

Style: [talking to camera / voiceover / text on screen]
Tone: [educational/funny/inspirational]
Include suggested text overlays if relevant.
```
