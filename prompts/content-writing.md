# Content Writing Prompts

## 1. Blog Post Outline

```
Write a blog post outline about "[topic]" for a [type of business] website.

Target audience: [who reads this]
Goal: [what you want readers to do after]
Word count target: [number]

Include:
- A catchy title (not clickbait)
- 5-7 subheadings
- 2-3 bullet points under each subheading
- A call to action at the end
```

## 2. Product Description

```
Write a product description for [product name].

Key features:
- [feature 1]
- [feature 2]
- [feature 3]

Target buyer: [who buys this]
Price point: [budget/mid-range/premium]
Tone: [casual/professional/luxury]

Keep it to 100-150 words. Focus on benefits, not just features.
```

## 3. Website Homepage Copy

```
Write homepage copy for [company name], a [what you do] based in [location].

We help [target audience] with [main services].

What makes us different: [your USP]

Include:
- A headline (under 10 words)
- A subheading (under 25 words)
- 3 short sections covering our main services
- A call to action

Tone: [describe your brand voice]
```

## 4. Case Study Write-Up

```
Turn these notes into a short case study (300-400 words):

Client: [name/industry]
Problem: [what they were struggling with]
What we did: [your solution]
Results: [numbers, outcomes]

Format: Problem → Solution → Results
Write it like a story, not a report.
```

## 5. Email Newsletter

```
Write a monthly email newsletter for [company name].

This month's updates:
- [update 1]
- [update 2]
- [update 3]

Include a tip or piece of advice related to [your industry].
Subject line options: give me 3 to choose from.
Keep the whole thing under 300 words.
```

## 6. Service Page Copy

```
Write copy for a service page about [service name].

What it is: [brief description]
Who it's for: [target customer]
How it works: [process in 3-4 steps]
Pricing: [starting from £X / quote-based / etc]

Include a section addressing common concerns or objections.
End with a clear CTA.
Word count: 400-500 words.
```

## 7. Meta Descriptions

```
Write meta descriptions for these pages:

1. [page title + URL]
2. [page title + URL]
3. [page title + URL]

Requirements:
- 150-160 characters each
- Include the main keyword naturally
- Make people want to click
- Don't start with "Welcome to" or "We are"
```
