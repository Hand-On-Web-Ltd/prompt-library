## How to Use

Copy any prompt, swap out the bits in `[square brackets]` with your own details, and paste it into ChatGPT, Claude, or whatever you're using.

Most of these work with any model — GPT-4, Claude, Gemini, Llama, etc.

## Tips

- Be specific. The more context you give, the better the output.
- Tell the AI who it is. "You are a customer service rep for a plumbing company" beats "write a reply".
- If the first output isn't great, ask it to try again with more detail.
